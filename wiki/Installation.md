# Installation

## Requirements

- Node.js `>=20.18.1`
- A Home Connect account with connected appliances
- A Home Connect developer application with a client ID
- Optionally a client secret if your app configuration uses one

## Install The Module

```bash
cd ~/MagicMirror/modules
git clone https://github.com/HeikoGr/MMM-HomeConnect2
cd MMM-HomeConnect2
npm install
```

Continue with [Quick Start](Quick-Start) after installation.
