# MMM-HomeConnect2 Wiki

MMM-HomeConnect2 connects Home Connect appliances to MagicMirror without requiring a local browser on the mirror device.

## Screenshot

![MMM-HomeConnect2](../img/screenshot.png)

Use this wiki if you want to:

- install the module cleanly
- register the required Home Connect developer app
- start with a minimal device-flow setup
- tune rendering and timeout options
- troubleshoot authentication, SSE, or rate-limit issues

## Typical Setup Path

1. Create a Home Connect developer application and copy its client ID.
2. Install the module.
3. Start with the example from [Quick Start](Quick-Start).
4. Complete the device-flow login from [Authentication](Authentication).
5. Adjust visibility and timing behavior in [Configuration](Configuration).
